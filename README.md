# Mimori-Tool
三森すずこ変換ツールです.

## 使い方
実行すると
```
何森?:
```
と聞かれますので, 作りたい三森すずこの数値(0~1000)を入力してください.
```
何森?: 3
```
結果が表示されます.
```
何森?: 3
三森すずこ
```
```
何森?: 4
四林林せぜこ
```

## C
### コンパイル
cフォルダの`mimori-tool.c`か`mimori-tool2.c`をgccでコンパイルしてください.
### 無印と2の違い
出力結果は同じですが, 実装方法が違います.
